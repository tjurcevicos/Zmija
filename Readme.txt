# Zmija

Klasična igra Zmija izrađena pomoću HTML-a, CSS-a i JavaScripta.

Cilj igre je skupljati voće i pritom izbjegavati sudar sa zidovima. Svakim skupljenim voćem zmija raste, a broj bodova se povećava.

===== Tehnologije =====

- HTML
- CSS
- JavaScript

===== Funkcionalnosti =====

- Upravljanje strelicama na tipkovnici
- Nasumično generiranje voća
- Rast zmije nakon skupljanja voća
- Brojanje bodova
- Game Over ekran
- Gumb za pokretanje nove igre

===== Kontrole =====

⬆️ Strelica gore  
⬇️ Strelica dolje  
⬅️ Strelica lijevo  
➡️ Strelica desno
